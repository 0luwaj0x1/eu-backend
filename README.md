# eu-backend

How to install the app 

```bash
npm install
```

To start the app

```bash
npm run start
```

List of available Endpoint

Question one

```bash
GET  /api/fullname   e.g  /api/malta
```

Question two

```bash
GET  /api/filter?words=a,b,z   
```

Question three

```bash
GET  /api/all  
```

Question four

```bash
GET  /api/spin
```
